# KLH ERP — Master Roadmap
> ตกลงกับคุณหนุ่ม 2026-06-06 — ทำทีละขั้น คุยทีละเมนู

## หลักการ
- คุยทีละเมนู ให้แต่ละเมนูทำงานครบถูกต้องก่อน
- เปลี่ยน UX/UI ใหม่ทั้งหมด **แต่ฟังก์ชันเดิมต้องทำงานเหมือนเดิม**

## ลำดับงาน
### ✅ Phase 0 (เพิ่งเสร็จ)
- สำรวจโครงสร้างชีต+เมนูทั้งหมด
- ข้อ 5 บัญชี: สต๊อกสิ้นเดือน KLH (stock_month) — สร้างแล้ว รอทดสอบ

### 🔲 ขั้นที่ 1: แก้ราคาโหลให้เสร็จ  ← กำลังทำ
- backend: getPriceListItems, posSearchProducts ส่ง retail_doz(AI)+barcode_doz(AH)
- frontend: แสดง/ขายราคาโหล ใน pricelist, pos, pos_pc, pos_t2

### 🔲 ขั้นที่ 2: คุยทีละเมนู (ตรวจฟังก์ชันให้ครบ)
- ไล่ทุกเมนู ยืนยันการทำงานถูกต้อง

### 🔲 ขั้นที่ 3: ระบบ WMS บริหารคลังสินค้า
- ตอนนี้ยังไม่ผูกกับใคร → ออกแบบ + เชื่อมกับ POS/สต๊อก/รับเข้า

### 🔲 ขั้นที่ 4: เปลี่ยน UX/UI ใหม่ทั้งระบบ
- ดีไซน์ใหม่ ฟังก์ชันเดิมทำงานเหมือนเดิม

### 🔲 ขั้นที่ 5: POS ผ่าน LINE OA
- Rich Menu + LIFF + ตะกร้า + QR + ตรวจสลิป + ออเดอร์เข้าระบบ

### 🔲 ขั้นที่ 6: ระบบบัญชีหลังบ้าน
- 6.1 บัญชีสำหรับภาษี (VAT ซื้อ/ขาย, ภพ.30)
- 6.2 บัญชีการเงินบริหาร: เจ้าหนี้ (AP) + ลูกหนี้ (AR)
- 6.3 งบกำไรขาดทุน (P&L)
- 6.4 งบดุล (Balance Sheet) — **ยังไม่ทำตอนนี้**
- 6.5 งบกระแสเงินสด (Cash Flow) — บางส่วน
